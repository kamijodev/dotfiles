# WezTerm

メインターミナルエミュレータ。リーダーキーベースの操作体系と IDE 風レイアウトスクリプトを備える。

## 設定ファイル

| ファイル | 用途 |
|---------|------|
| `~/.config/wezterm/wezterm.lua` | メイン設定（外観、フォント、マウス） |
| `~/.config/wezterm/keybinds.lua` | キーバインド定義 |
| `~/.local/bin/ide` | IDE 風レイアウト作成スクリプト |

## 外観

- フォント: Hack Nerd Font (12.0)
- カラースキーム: Gruvbox Material (Gogh)
- 背景透過: 0.95
- タブバー: 複数タブ時のみ表示、三角形装飾付き
  - アクティブタブ: 紫 (#862aa1)
  - インアクティブタブ: ダーク (#282c34)

## リーダーキー

**Ctrl+S**（タイムアウト 2秒）

リーダーキーを押した後、続くキーでコマンドを実行する。デフォルトキーバインドは無効化済み。

## キーバインド

### タブ操作

| キー | 動作 |
|------|------|
| Leader + T | 新規タブ |
| Ctrl+Tab | 次タブ |
| Ctrl+Backspace | 前タブ |
| Leader + { | タブを左に移動 |
| Leader + } | タブを右に移動 |
| Super+W | タブを閉じる（確認付き） |

### ペイン操作

| キー | 動作 |
|------|------|
| Leader+Ctrl+D | 垂直分割 |
| Leader+Ctrl+V | 水平分割 |
| Leader+Ctrl+/ | ペインを閉じる（確認付き） |
| Leader+Ctrl+H/L/K/J | ペイン間フォーカス移動 |
| Leader+Z | ペインズーム（トグル） |
| Leader+S → H/L/K/J | ペインサイズ調整モード |
| Leader+A → H/L/K/J | ペイン選択モード（1秒タイムアウト） |

### ワークスペース

| キー | 動作 |
|------|------|
| Leader + W | ワークスペース切り替え |
| Leader + Shift+W | 新規ワークスペース |
| Leader + $ | ワークスペース名前変更 |

### コピー・その他

| キー | 動作 |
|------|------|
| Leader + [ | コピーモード開始（Vim 操作） |
| Super+C | コピー |
| Super+V / Ctrl+V | 貼り付け |
| Super+P / Ctrl+Shift+P | コマンドパレット |
| Alt+Enter | フルスクリーン切り替え |
| Alt+J/K | 2行スクロール |
| Ctrl++/- | フォントサイズ増減 |
| Ctrl+0 | フォントサイズリセット |

### コピーモード

Leader+[ で開始。Vim ライクな操作でテキスト選択・コピー。

- 移動: H/J/K/L、W/B/E（単語）、^/$（行頭/末）、G/gg（末尾/先頭）
- 選択: V（文字）、Ctrl+V（ブロック）、Shift+V（行）
- コピー: Y（コピー）、Enter（コピーして終了）
- 終了: Escape / Ctrl+C / Q

## IDE レイアウトスクリプト

`~/.local/bin/ide` を実行すると、WezTerm のペインを以下のレイアウトに分割する。

```
+------------------+--------+
|                  |   AI   |
|      Main        |  Chat  |
|                  |        |
+--------+---------+--------+
| Term 1 | Term 2  |
+--------+---------+
```

- Main: メインエディタ領域
- AI Chat: 右 25%（Claude Code 等）
- Term 1/2: 下 20%（ターミナル2分割）
